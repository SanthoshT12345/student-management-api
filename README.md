A simple RESTful API built using Node.js and Express to perform CRUD operations on student data.
This project was developed as part of a MERN stack internship to understand backend workflows and API testing using Postman.
## Features
- Create a new student
- View all students
- Update student details using ID
- Update student details using name
- Delete a student
## Tech Stack
- Node.js
- Express.js
- Postman (for API testing)
